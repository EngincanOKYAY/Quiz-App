This is a quiz application.
There is 1 question and 4 answers at a time.
When you mark the correct answer, the green color appears, and when you mark the wrong answer, the red color appears.
Once one option is selected, the other options cannot be marked.
When an option is selected, the "next" button appears.
You can move on to the next question using the "next" button.
You will be given your score at the end of the quiz.
HTML, CSS and JavaScript were used while making this application.
